Установка приложения

Это приложение можно установить на ваше устройство для быстрого доступа:

iOS (Safari)

  1. Нажмите кнопку «Share» (поделиться)
  2. Выберите «Add to Home Screen» (На экран «Домой»)
  3. Нажмите «Add» (Добавить)

iOS (Chrome)

  1. Нажмите на три точки в верхнем правом углу
  2. Прокрутите меню вниз и выберите «Добавить на экран «Домой»
  3. Нажмите «Добавить»

Android (Chrome)

  1. Нажмите на три точки в меню
  2. Выберите «Install app»
  3. Нажмите «Install»

О приложении

Это приложение разработано с учетом вашей приватности:

  • Все ваши данные хранятся только локально на вашем устройстве
  • Приложение работает без подключения к интернету
  • Никакие данные не отправляются на сервер
  • Исходный код приложения опубликован на GitHub и доступен для проверки
Важно: все данные хранятся локально в браузере. При очистке данных браузера ваши записи будут удалены.
